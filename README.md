# Local_Meteo

Pipeline meteo modulare per Home Assistant.  
Raccoglie dati da DPC, ARPAV e Open-Meteo, calcola condizioni del cielo, temperatura, umidità, vento e pioggia, e li pubblica su Home Assistant.
## Configurazione
Vedi `config/config.yaml` per impostare latitudine, longitudine, servizi attivi e parametri.
